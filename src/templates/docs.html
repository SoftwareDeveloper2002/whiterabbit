WhiteRabbit Framework Documentation

Introduction

WhiteRabbit is a lightweight Python web framework for building fast and simple web applications with routing, template rendering, static file handling, and middleware support.

Installation

git clone https://github.com/yourusername/whiterabbit.git
cd whiterabbit
pip install -r requirements.txt

Project Structure


whiterabbit/
    __init__.py
    app.py
    router.py
    response.py
    templates.py
    middleware.py
    
src/
    app.py
    routes/
        landing.py
        docs.py
    components/
        navbar.py
    templates/
        index.html
        docs.html
    static/
        style.css

Getting Started

from whiterabbit import App
from src.routes import landing, docs

app = App()

if __name__ == "__main__":
    app.run()

Routing

from whiterabbit import app
from whiterabbit.components.navbar import Navbar

@app.route("/")
def landing(request):
    navbar = Navbar()
    return app.render_template("index.html", {"navbar": navbar.render()})

Templates

Use {{ variable }} in your HTML templates for dynamic content.

Static Files

Store CSS, JS, images inside src/static/ and reference them using /static/filename.

Components

Create reusable components inside src/components/ and render them in templates.

Running the App

python -m src.app
# Open http://127.0.0.1:8000
