A lightweight Python web framework for building fast, simple, and maintainable web applications.

Why Use WhiteRabbit?

Simple Routing

Define routes easily using decorators. No complicated configuration is needed.

Template Rendering

Use HTML templates with dynamic content injection using {{ variable }} syntax.

Middleware Support

Add middleware functions to handle requests and responses globally.

Static Files & Components

Serve static files like CSS/JS and create reusable UI components easily.

Quick Start

Get your first WhiteRabbit application running in minutes:

from whiterabbit import App
from src.routes import landing, docs

app = App()

if __name__ == "__main__":
    app.run()  # Open http://127.0.0.1:8000

Then create routes in src/routes/ and templates in src/templates/.
